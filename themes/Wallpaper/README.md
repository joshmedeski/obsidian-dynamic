# Obsidian Wallpaper Theme
An obsidian theme to showcase wallpapers with minimal UI.

> [!NOTE]
> This theme is designed to be used with the [Obsidian Wallpaper Plugin](#)

## Features
- Semi-transparent view (to use with obsidian dynamic wallpaper theme)
- Display a user chosen wallpaper in the background (with the plugin)

## Ideas
- Dynamic color schemes
- Color scheme generator UI
